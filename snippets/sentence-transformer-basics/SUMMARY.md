# Sentence Transformer Basics

A minimal demonstration of text embedding and semantic similarity using the sentence-transformers library. This project shows how to encode text queries and documents into vector embeddings and calculate their semantic similarity scores.

## Primary Technologies

- **Python 3.13+** - Programming language
- **sentence-transformers** - Library for state-of-the-art text embeddings
- **uv** - Fast Python package and project manager
- **Ruff** - Python linter and formatter
- **HuggingFace Models** - Pre-trained transformer models for embeddings

## Key Features

- Load embedding models from HuggingFace or local paths
- Encode text into high-dimensional vectors
- Calculate semantic similarity between queries and documents
- Simple API demonstration with practical examples
