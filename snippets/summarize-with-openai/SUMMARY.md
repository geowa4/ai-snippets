# Summarize with OpenAI

This project is a Python utility that automatically generates executive summaries from incident reports using OpenAI language models.
It processes Markdown incident reports stored in the `incident-reports` directory and produces concise, active-voice summaries in the `incident-summaries` directory.

## Technologies Used

- **Python 3.13+**: Core programming language
- **OpenAI API**: For language model access
- **pydantic-ai**: Framework for AI agent integration
- **uv**: Modern Python package manager
- **ruff**: Python linter and formatter
- **mypy**: Static type checker
