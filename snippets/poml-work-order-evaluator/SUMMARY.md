# POML Work Order Evaluator

## Purpose

A demonstration of Microsoft's POML (Prompt Orchestration Markup Language) for structured prompt engineering. This snippet showcases how to evaluate IT work orders using POML's core components: `<role>`, `<example>`, `<list>`, and `<stepwise-instructions>`. The example creates a maintainable, reusable template that assesses work orders for thoroughness, documented steps, and transactional nature, illustrating best practices for LLM prompt construction.

## Key Technologies

- **[POML](https://microsoft.github.io/poml/)** - Prompt Orchestration Markup Language for structured, maintainable LLM prompts
- **Python 3.13+** - Modern Python with type hints and async support
- **uv** - Fast Python package manager for dependency management
